## OBJETIVOS DEL PROYECTO

*"CHAKRAFILAMENT" busca transformar residuos agrícolas amazónicos en materiales avanzados para impresión 3D, utilizando nanocelulosa extraída de la chakra (Plukenetia volubilis), una planta nativa de la región amazónica de Ecuador. Esta iniciativa aborda la demanda de filamentos biodegradables que reduzcan la dependencia de plásticos sintéticos, fomentando la innovación en manufactura aditiva sostenible.*

*Objetivo principal: Desarrollar filamentos de alta calidad para impresión 3D a partir de nanocelulosa extraída de la chakra amazónica, validando su viabilidad técnica y mecánica.*

*1. Impulsar la aplicación de principios de economía circular mediante planificación estratégica que integre extracción, procesamiento y reciclaje de subproductos vegetales.*

*2. Proporcionar una gama ampliada de materiales orgánicos y sustentables, mejorando el rendimiento (resistencia, flexibilidad y biodegradabilidad) para investigadores, diseñadores y fabricantes de productos impresos en 3D.*

// Scroll Animation
function revealOnScroll() {
  const reveals = document.querySelectorAll(".reveal");

  reveals.forEach(element => {
    const windowHeight = window.innerHeight;
    const elementTop = element.getBoundingClientRect().top;
    const elementVisible = 100;

   if (elementTop < windowHeight - elementVisible) {
  element.classList.add("active");
  }
  });
}

window.addEventListener("scroll", revealOnScroll);

// Counter animation
const counters = document.querySelectorAll(".counter");

counters.forEach(counter => {
  const updateCount = () => {
    const target = +counter.getAttribute("data-target");
    const count = +counter.innerText;
    const increment = target / 200;

   if (count < target) {
   counter.innerText = Math.ceil(count + increment);
  setTimeout(updateCount, 10);
  } else {
   counter.innerText = target;
  }
  };

  updateCount();
});
